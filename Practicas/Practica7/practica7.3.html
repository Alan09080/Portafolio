HTML
El lenguaje de marcado estándar utilizado para crear páginas web.
CSS
El lenguaje utilizado para dar estilo a las páginas web creadas con HTML.
JavaScript
Un lenguaje de programación utilizado para agregar interactividad y dinamismo a las páginas web.
DOM
El Document Object Model es una interfaz de programación que representa la estructura de un documento HTML y permite acceder y manipular sus elementos mediante código JavaScript.
HTTP
El protocolo de transferencia de hipertexto es el protocolo utilizado para la comunicación entre clientes y servidores web en Internet.
FTP
El protocolo de transferencia de archivos es un protocolo utilizado para la transferencia de archivos entre computadoras conectadas a una red.
URL
La dirección web que se utiliza para acceder a un recurso en Internet.
SEO
La optimización de motores de búsqueda es el proceso de mejorar la visibilidad de un sitio web en los resultados de búsqueda de motores de búsqueda como Google.
CMS
Un sistema de gestión de contenidos es una herramienta que permite crear y administrar fácilmente contenido para sitios web, sin necesidad de conocimientos avanzados de programación.
PHP
Un lenguaje de programación utilizado principalmente para el desarrollo de aplicaciones web del lado del servidor.
